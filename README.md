# Dissecting OpenClaw: Under the Hood of a Self-Evolving AI Agent

A presentation by Giovanni Sakti, built with [Slidev](https://sli.dev).

## Prerequisites

- Node.js 18+

## Setup

```bash
npm install
```

## Presenting

Start the dev server:

```bash
npm run dev
```

Open http://localhost:3030 in your browser. Press `f` for fullscreen.

### Presenter mode

Navigate to http://localhost:3030/presenter for speaker notes and slide preview.

### Keyboard shortcuts

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `f` | Toggle fullscreen |
| `o` | Slide overview |
| `d` | Toggle dark mode |

## Export

Build as static SPA:

```bash
npm run build
```

Export to PDF (requires [playwright-chromium](https://npmjs.com/package/playwright-chromium)):

```bash
npm run export
```
